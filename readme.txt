A simple way to insert custom codes into the header, body, and footer of your WordPress site.

== Description ==
The **Simply Insert Headers and Footers** plugin allows you to easily add custom code snippets to your WordPress site. You can insert code in the header, body, or footer sections without modifying your theme files directly. This is useful for adding tracking scripts, custom styles, or any other third-party code.

== Features ==

- Add custom code to the header, body, and footer.
- Easy-to-use interface in the WordPress admin dashboard.
- Code sanitization to prevent XSS vulnerabilities.
- Safe output encoding to ensure security.

== Installation ==

1. Download the plugin zip file.
2. Go to your WordPress admin panel.
3. Navigate to **Plugins > Add New**.
4. Click on **Upload Plugin** and select the downloaded zip file.
5. Click **Install Now** and then activate the plugin.

Alternatively, you can install the plugin from the WordPress Plugin Directory by searching for "Simply Insert Headers and Footers".

== Usage ==

1. After activating the plugin, navigate to **Insert Headers & Footers** in the WordPress admin menu.
2. You will find three textarea fields for Header Code, Body Code, and Footer Code.
3. Enter your custom code snippets in the respective fields.
4. Click the **Save Changes** button to apply.

== Frequently Asked Questions ==

= Can I use HTML in the code fields? =

Yes, you can use HTML in all code fields. The plugin sanitizes the input to ensure that only safe HTML is allowed.

= What types of code can I insert? =

You can insert any type of code that is valid HTML or JavaScript. Common uses include tracking scripts, CSS styles, and other third-party integrations.

= Is this plugin secure? =

Yes, the plugin includes sanitization for input and escaping for output to help protect against XSS vulnerabilities.
